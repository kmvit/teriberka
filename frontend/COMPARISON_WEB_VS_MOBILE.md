# Что отсутствует в мобильном приложении Teriberka

## Контентные разделы

- Блог «О Териберке» — список статей, категории, рекомендуемые
- FAQ — страницы вопросов и ответов

## Авторизация и регистрация

- Регистрация по телефону (SMS-код + reCAPTCHA)
- Вход по телефону (сейчас только по email)
- Роль «Гостиница» при регистрации (есть: клиент, владелец катера, гид)
- Защита от брутфорса (блокировка 5 мин после 3 неудачных попыток)

## Профиль

- Интеграция с Telegram — подключение/отключение уведомлений, статус (`getTelegramStatus`, `unlinkTelegram`)

## Бронирование

- Промокоды — поле ввода, превью скидки, учёт при расчёте
- Бронирование от лица гостиницы — режим «Забронировать для гостя», кешбэк
- Ссылка на полную оплату для гостиниц (`create_full_payment_link`)
- Блокировка мест (`block_seats`, `blocked_seats`, `unblock`)

## Управление судами (для владельцев)

- Выбор причала (docks) при создании и редактировании судна
- Отображение причала на главной, в деталях рейса и в бронированиях
- Ссылка на Яндекс.Карты для причала (`dock.yandex_location_url`)

## Бронирования в профиле

- Режим для гостиницы — отдельный список, кешбэк, ссылка на полную оплату
- Кнопка «Создать ссылку на полную оплату» для отелей

## Админ-панель

- Управление капитанами (`/profile/admin/captains`)
- Финансовая таблица капитанов с фильтрацией по капитану и периоду

## Оплата

- Страницы результата оплаты (`/payment/success`, `/payment/fail`)
- Проверка статуса платежа (`paymentsAPI.checkStatus`)

## Прочее

- Переключатель языка RU/CH в навбаре
- Использование API настроек сайта (`v1/site/settings/current`)
